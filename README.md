# Projeto Café Expresso ☕

## Descrição
Este projeto simula o sistema de uma máquina de Café Expresso, desenvolvido como requisito da disciplina. O foco principal do desenvolvimento é a aplicação de boas práticas de programação e a obrigatoriedade de cobertura com testes unitários para todas as funcionalidades implementadas.

## Funcionalidades (Escopo Inicial)
* Seleção de diferentes tipos de bebidas (ex: Expresso, Cappuccino, Mocaccino).
* Verificação e consumo de ingredientes (água, grãos de café, leite).
* Sistema simples de cobrança/pagamento.

## Tecnologias Utilizadas
